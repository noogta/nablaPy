===========================================
Instructions d'installation et d'utilisation
===========================================

Bienvenue !

Ce projet nécessite l'installation de certaines bibliothèques Python ainsi que de modules externes pour fonctionner correctement. Suivez les instructions ci-dessous pour configurer l'environnement et exécuter le programme avec succès.

1. Prérequis
   - Python : Assurez-vous d'avoir Python installé sur votre système. Le projet est compatible avec Python 3. Vous pouvez vérifier la version de Python installée en exécutant la commande suivante dans le terminal :
     ```
     python --version
     ```
   - pip : Vérifiez que vous avez pip installé. pip est le gestionnaire de packages Python et est généralement installé avec Python. Vous pouvez vérifier la version de pip en exécutant la commande suivante dans le terminal :
     ```
     pip --version
     ```
2. Chemin de votre interpréteur Python
  Vous devez mettre dans le fichier Makefile, le chemin où se trouve votre interpréteur python

3. Installation des bibliothèques Python
   Vérifier que vous avez la commande make sur votre ordinateur. Éxécutez la commande suivante dans un terminal ouvert depuis le dossier:
    make
